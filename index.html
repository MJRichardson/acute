Acute

The ubiquity of HTML with the productivity of C#.

Acute is a framework for developing client-side HTML applications in C#.

Acute allows you to write MV* applications, using HTML for the views, and C# for the controllers, models, view-models, services, and whatever other components/patterns you desire. The C# portions are compiled into javascript, allowing your application to run in browsers everywhere.

Acute is inspired by AngularJS (and utilizes it for implementation). Acute aims to pair the industrial-grade concepts of AngularJS (templates, controllers, dependency-injection, etc) with an industrial-grade language (C#).
